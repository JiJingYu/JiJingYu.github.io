这是我的新博客

今年研二了，明年3月就要研究生毕业了。
回顾研究生阶段，个人成长还是很大的。

  1. 工作方面，顺利拿到了地平线机器人的实习offer；
  2. 科研方面，发表了几篇文章，虽然受限于各种客观条件，没能发表过计算机视觉领域三大会议的paper，但是也在子领域的顶级期刊上发表了一些文章，总体来看研究生阶段没有虚度。
  3. 管理方面，从单打独斗到能带领小团队做事，积累了一些经验。带领研究生发文章，指导本科生做项目，学习“宁向东的清华管理学课”等。
  4. 运动方面，顺利减肥，从小胖子到能跑完半马；
  5. 金融方面，试水炒了炒股，从大A股到美股港股到数字货币，从“薛兆丰的经济学课程”到“香帅的北大金融学课程”，了解了不少经济学与金融学知识。

博客园( http://www.cnblogs.com/nwpuxuezha/ ) 暂时弃用，以后转到github上分享个人成长的收获。
2018年，期待自己有更多的成长。
