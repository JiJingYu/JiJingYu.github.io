前言

从拿到新电脑到安装 PyTorch 0.4
深度学习框架的安装越来越傻瓜式了。

简单记录下自己安装 PyTorch 的流程。

硬件配置：

  1. GPU：GTX1080Ti x 4
  2. CPU：E5 2665 x2, 16C 32T
  3. 内存：128 GB
  4. 显示器：4K 渣显示器一台

操作系统：

Ubuntu 16.04

安装过程：

install anaconda in new computer

从这里下载最新版 anaconda

change conda source

install pytorch

由于 pip 获取的库通常比 conda 更新一些，所以我使用 pip 进行 python 包的安装。

install ipython3

install pycharm

从这里下载最新版 pycharm

通常免费的社区版就够用了，土豪可以使用专业版。
