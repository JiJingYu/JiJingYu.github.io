hexo+github 换了新电脑怎么办

hexo+github 换了新电脑怎么办

今天写一篇流水账。

准备工作

  1. 把之前写的 md 文件都备份一下。
  2. 新电脑安装 Github，安装 hexo

配置博客

  1. 新建目录，例如 jingyu.ji.blog

  2. 在该目录下运行 hexo init

  3. 修改 _config.yml 文件，

  4. 部署 Git：npm install hexo-deployer-git --save

导入博客并上传

  1. 拷贝 CNAME 文件到 ./source/ 目录
  2. 拷贝 .md 格式的博客文件到 ./source/_posts 目录
  3. hexo clean -->hexo generate -->hexo deploy ，或者直接 hexo d -g， 大功告成
